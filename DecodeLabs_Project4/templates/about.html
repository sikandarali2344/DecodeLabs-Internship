👁️ Project 4: Computer Vision

Project 4 is part of the DecodeLabs Industrial Training Kit for Batch 2026.

This project demonstrates Computer Vision capabilities including:

  • 📖 Optical Character Recognition (OCR) using Tesseract
  • 🎯 Object Detection using MobileNet-SSD
  • ⚙️ Image Pre-Processing (Grayscale, Blur, Thresholding)
  • 📊 Confidence Scoring & Analysis
  • 🖼️ Visual Confirmation & Bounding Boxes

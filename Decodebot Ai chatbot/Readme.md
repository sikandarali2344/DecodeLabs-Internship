
# 🤖 DecodeBot AI - Intelligent Chatbot Assistant

## 🚀 Project Overview

**DecodeBot AI** is an advanced, AI-powered chatbot application built with **Streamlit** and integrated with **DeepSeek R1** (70B parameters) via **Groq API**. It provides intelligent responses, code generation, explanations, and problem-solving capabilities through a modern, dark-mode interface.

---

## ✨ Key Features

### 🤖 AI Capabilities
- **Code Generation** - Write clean, documented code in Python, JavaScript, HTML/CSS
- **Code Explanation** - Line-by-line breakdown of complex code
- **Debugging Help** - Identify and fix errors quickly
- **Concept Teaching** - Learn programming with examples and analogies
- **Problem Solving** - Solve algorithmic and coding challenges

### 💬 Chat Features
- Real-time AI responses with typing animation
- Save and load conversation history
- Export chats as JSON files
- Clear history with one click
- Multi-language support (English, Hindi, Gujarati)

### 🎨 User Interface
- Professional dark mode design
- Styled chat bubbles (User vs Bot)
- Responsive dashboard with metrics
- Circular logo integration
- Smooth animations and transitions

### 🔐 Security & Management
- User authentication (Signup/Login)
- Session management
- Local chat storage
- Environment variable protection

## 📦 Quick Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/DecodeBot-AI.git
cd DecodeBot-AI

# Install dependencies
pip install -r requirements.txt

# Set up environment variables
echo "API_KEY=your_groq_api_key_here" > .env

# Run the application
streamlit run main.py
```
